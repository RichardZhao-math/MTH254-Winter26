    Subsection Vectors in \(\boldsymbol{\mathbb{R}^2}\)

    In case you are not familiar with the notation \(\mathbb{R}^2\text{,}\) it simply means the set of all ordered pairs of real numbers. Symbolically speaking,
    \begin{equation*} \mathbb{R}^2 = \mathbb{R}\times \mathbb{R} = \left\{(x,y) \mid x,y\in \mathbb{R}\right\} \end{equation*}
    That is, we will work with vectors in two dimensions in this section.
    ðŸ”—
    Letâ€™s first define what a vector is in \(\mathbb{R}^2\text{.}\) A vector in \(\mathbb{R}^2\) is an object determined by two points in the plane: an initial point \(P\) (the tail) and a terminal point \(Q\) (the tip). We write
    \begin{equation*} \mathbf{v} = \overrightarrow{PQ} \end{equation*}
    and we draw \(\mathbf{v}\) as an arrow pointing from \(P\) to \(Q\text{.}\)
    ðŸ”—
    In \(\mathbb{R}^2\text{,}\) each point is represented by an ordered pair of real numbers. Then we can define the components of a vector using the ordered pairs of the two points as follows.
    ðŸ”—

    Definition 12.1.2. Components of a Vector.

    The components of \(\mathbf{v} = \overrightarrow{PQ}\text{,}\) where \(P = (a_1, b_1)\) and \(Q = (a_2, b_2)\text{,}\) are the quantities
    \begin{equation*} a = a_2 - a_1 \quad \text{($x$-component)}\, , \qquad \qquad b = b_2 - b_1 \quad \text{($y$-component)} \end{equation*}
    The pair of components is denoted \(\langle a,b \rangle\text{.}\)

    Subsection The Attributes of Vectors

    When defining a vector, it is important to note its magnitude and the direction.
    ðŸ”—
    The magnitude of a vector \(\mathbf{v}\) is the distance from the tail to the tip (aka the length of the vector). This is denoted as \(\|\mathbf{v}\|\text{.}\)
    ðŸ”—
    Given a vector \(\mathbf{v} = \langle a,b \rangle\text{,}\) we can determine its magnitude using the Pythagorean Theorem, as demonstrated in the figure below.
    ðŸ”—
    By the Pythagorean Theorem, we have
    \begin{align*} \|\mathbf{v}\| \amp= \sqrt{(a_1 - a_2)^2 + (b_1 - b_2)^2} \\ \amp= \sqrt{a^2 + b^2} \end{align*}
    ðŸ”—
    The magnitude of a vector is really called the \(\boldsymbol{\ell^2}\) norm or the Euclidean norm of a vector in more advanced math courses. In general, norms must satisfy the following three properties:
    1. \(\|\mathbf{v}\| \geq 0\text{,}\) with \(\|\mathbf{v}\| = 0\) if and only if \(\mathbf{v} = \mathbf{0}\text{.}\)
      ðŸ”—
      ðŸ”—
    2. \(\| \lambda\mathbf{v}\| = |\lambda|\|\mathbf{v}\|\) for all scalars \(\lambda \in \mathbb{R}\text{.}\)
      ðŸ”—
      ðŸ”—
    3. \(\|\mathbf{v} + \mathbf{w}\| \leq \|\mathbf{v}\| + \|\mathbf{w}\|\) for any two vectors \(\mathbf{v}\) and \(\mathbf{w}\text{,}\) with equallity only if \(\mathbf{v} = 0\text{,}\) \(\mathbf{w} = 0\text{,}\) or if \(\mathbf{w} = \lambda\mathbf{v}\) where \(\lambda \geq 0\text{.}\) (This is the famous triangle inequality).
      ðŸ”—
      ðŸ”—
    This isnâ€™t a linear algebra or functional analysis class, so we wonâ€™t go deeper into norms here. If you are interested, feel free to do more digging on your own (or ask Richard!).
    ðŸ”—
    Now letâ€™s discuss the direction of a vector. The direction of a vector tells us where the vector is pointing. We can indicate the direction of a vector by using the vector itself. As you imagine, the magnitude of a vector has nothing to do with its direction, so we sometimes use a unit vector, which is a vector of length \(1\text{,}\) to indicate the direction, when it is not necessary to specify length.
    ðŸ”—
    If we have a unit vector whose tail is at the origin, then its tip lies on the unit circle. We usually denote a unit vector by \(\mathbf{e}\text{,}\) defined as
    \begin{equation*} \mathbf{e} = \langle \cos(\theta), \sin(\theta)\rangle \end{equation*}
    where \(\theta\) is the angle between \(\mathbf{e}\) and the positive \(x\)-axis, indicated in the figure below.
    ðŸ”—
    But what if we are given an arbitrary non-zero vector and we want to find a unit vector in the same direction? One way to do so is to divide the vector by its magnitude (multiplying a positive scalar doesnâ€™t change the direction of a vector). That is, given \(\mathbf{v}\text{,}\) the unit vector in the direction of \(\mathbf{v}\) is
    \begin{equation*} \mathbf{e_v} = \frac{1}{\|\mathbf{v}\|}\mathbf{v} \end{equation*}
    ðŸ”—

    A quick proof to show that \(\mathbf{e_v}\) is a unit vector.

    Let \(\mathbf{v}\) be a non-zero vector. Then \(\|\mathbf{v}\| \gt 0\text{.}\) This implies that
    \begin{align*} \|\mathbf{e_v}\| \amp= \left\|\frac{1}{\|\mathbf{v}\|}\mathbf{v}\right\| \\ \amp= \left|\frac{1}{\|\mathbf{v}\|}\right| \|\mathbf{v}\| \amp\amp\text{by the second norm property} \\ \amp= \frac{1}{\|\mathbf{v}\|}\|\mathbf{v}\| \amp\amp\text{since } \frac{1}{\|\mathbf{v}\|} \gt 0 \\ \amp= 1 \end{align*}
    Hence, \(\mathbf{e_v}\) is a unit vector.
