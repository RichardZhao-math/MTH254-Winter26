    Section 14.2 Limits and Continuity in Several Variables

    In our previous study of functions of a single variable, the concept of a limit was straightforward in one respect. We could only approach a point \(x=a\) from two directions, the left or the right. If the function approached the same value from both sides, the limit existed.
    ðŸ”—
    Now that we are working with functions of two variables, the domain is a plane. This means we can approach a point \((a,b)\) from infinitely many directionsâ€”along the \(x\)-axis, the \(y\)-axis, a diagonal line, a parabola, or even a spiral. This increase in freedom makes the definition of a limit more complex but also more interesting. In this section, we will learn how to determine if these limits exist and how to evaluate them.
    ðŸ”—

    Objectives

    After this section, students will be able to:
    ðŸ”—
    • understand the definition of a limit for functions of two variables informally.
      ðŸ”—
      ðŸ”—
    • evaluate limits for function of two variables.
      ðŸ”—
      ðŸ”—
    • prove that a limit does not exist.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    Recall from MTH 251Z (or MH 251) that limits allow us to describe a functionâ€™s behavior near a specific point, even if the function is not defined at that point. The fundamental idea is that we can make the functionâ€™s output arbitrarily close to a limit value \(L\) by choosing inputs that are sufficiently close to the target point.
    ðŸ”—
    The idea of limits also extends to functions of several variables.
    ðŸ”—
    Below is the formal definition of limits for functions of two variables.
    ðŸ”—

    Definition 14.2.2. Limit.

    Assume that \(f(x,y)\) is defined near \(P = (a,b)\text{.}\) Then
    \begin{equation*} \lim_{(x,y) \to P} f(x,y) = L \end{equation*}
    if, for any \(\varepsilon \gt 0\text{,}\) there exists \(\delta \gt 0\) such that if \((x,y)\) satisfies
    \begin{equation*} 0 \lt d\lp (x,y), P \rp \lt \delta \, , \quad \text{then} \, |f(x,y) - L| \lt \varepsilon \end{equation*}
    ðŸ”—
    ðŸ”—
    This is a calculus class, not a real analysis class, so we wonâ€™t be doing any epsilon-delta proofs in this class. Make sure you are comfortable with the limit idea informally. That is, the limit of a function at a point is the value that the function approaches as we get closer and closer to the point.
    ðŸ”—
    Recall back in MTH 251Z (or MTH 251), a function is continuous at a point if the limit of the function at that point is equal to the value of the function at that point. This is the same for functions of several variables.
    ðŸ”—

    Definition 14.2.3. Continuity.

    A function \(f\) of two variables is continuous at \(P = (a,b)\) if
    \begin{equation*} \lim_{(x,y) \to (a,b)} f(x,y) = f(a,b) \end{equation*}
    We say that \(f\) is continuous if it is continuous at each point \((a,b)\) in its domain.
    ðŸ”—
    ðŸ”—
    That is, if we know that a function is continuous at a point (no hole, no jump, no asymptote), then we can evaluate the limit of the function at that point by directly substituting the point into the function.
    ðŸ”—
    Also recall back in MH 251Z (or MTH 251), a function is continuous at a point if the building blocks of the function are continuous at that point AND the function value is defined at that point.
    ðŸ”—

    Example 14.2.4.

    Evaluate the limit \(\ds \lim_{(x,y) \to (-2,1)} \dfrac{2x^2}{4x + y}\text{.}\)
    ðŸ”—
    Hint.
    Try arguing that the function \(f\) is continuous at \((-2,1)\text{.}\) Then you can evaluate the limit by directly substituting the point into the function.
    ðŸ”—
    ðŸ”—
    Solution.
    Observe that \(z = 2x^2\) and \(z = 4x + y\) are both continuous at \((-2,1)\text{.}\) Also, the denominator \(4x + y\) is not equal to 0 at \((-2,1)\text{.}\) Hence, \(f\) is continuous at \((-2,1)\text{.}\)
    ðŸ”—
    Thus, we can evaluate the limit by directly substituting the point into the function. That is,
    \begin{align*} \lim_{(x,y) \to (-2,1)} \dfrac{2x^2}{4x + y} \amp= \dfrac{2(-2)^2}{4(-2) + 1} \\ \amp= -\dfrac{8}{7} \end{align*}
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Again, back in MTH 251Z (or MTH 251), we also learned about the basic limit laws. They also hold for limits of functions of several variables.
    ðŸ”—

    Theorem 14.2.5. Limit Laws.

    Assume that \(\ds \lim_{(x,y) \to P}f(x,y)\) and \(\ds \lim_{(x,y) \to P}g(x,y)\) exist. Then
    1. Sum Law:
      \begin{equation*} \lim_{(x,y) \to P} \lp f(x,y) + g(x,y) \rp = \lim_{(x,y) \to P} f(x,y) + \lim_{(x,y) \to P} g(x,y) \end{equation*}
      ðŸ”—
      ðŸ”—
    2. Constant Multiple Law: For any number \(k\text{,}\)
      \begin{equation*} \lim_{(x,y) \to P} k f(x,y) = k \lim_{(x,y) \to P} f(x,y) \end{equation*}
      ðŸ”—
      ðŸ”—
    3. Product Law:
      \begin{equation*} \lim_{(x,y) \to P} f(x,y) g(x,y) = \lp \lim_{(x,y) \to P} f(x,y) \rp \lp \lim_{(x,y) \to P} g(x,y) \rp \end{equation*}
      ðŸ”—
      ðŸ”—
    4. Quotient Law: If \(\ds \lim_{(x,y) \to P} g(x,y) \neq 0\text{,}\) then
      \begin{equation*} \lim_{(x,y) \to P} \dfrac{f(x,y)}{g(x,y)} = \dfrac{\ds \lim_{(x,y) \to P} f(x,y)}{\ds \lim_{(x,y) \to P} g(x,y)} \end{equation*}
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 14.2.6.

    Assume that \(\ds \lim_{(x,y) \to (2,5)} f(x,y) = 3\) and \(\ds \lim_{(x,y) \to (2,5)} g(x,y) = 7\text{.}\) Find the following limits.
    1. \(\displaystyle \ds \lim_{(x,y) \to (2,5)} \lp g(x,y) - 2f(x,y) \rp\)
      ðŸ”—
      ðŸ”—
    2. \(\displaystyle \ds \lim_{(x,y) \to (2,5)} f(x,y)^2g(x,y)\)
      ðŸ”—
      ðŸ”—
    3. \(\displaystyle \ds \lim_{(x,y) \to (2,5)} \dfrac{f(x,y)}{f(x,y) + g(x,y)}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    Solution.
    1. \begin{align*} \lim_{(x,y) \to (2,5)} \lp g(x,y) - 2f(x,y) \rp \amp= \lim_{(x,y) \to (2,5)} g(x,y) + \lim_{(x,y) \to (2,5)} -2f(x,y) \amp\amp\text{by Sum Law}\\ \amp= \lim_{(x,y) \to (2,5)} g(x,y) - 2\lim_{(x,y) \to (2,5)} f(x,y) \amp\amp\text{by Constant Multiple Law}\\ \amp= 7 - 2(3) \\ \amp= 1 \end{align*}
      ðŸ”—
      ðŸ”—
    2. \begin{align*} \lim_{(x,y) \to (2,5)} f(x,y)^2g(x,y) \amp= \lp \lim_{(x,y) \to (2,5)} f(x,y) \rp^2 \lp \lim_{(x,y) \to (2,5)} g(x,y)\rp \amp\amp\text{by Product Law}\\ \amp= 3^2 \cdot 7 \\ \amp= 63 \end{align*}
      ðŸ”—
      ðŸ”—
    3. \begin{align*} \lim_{(x,y) \to (2,5)} \dfrac{f(x,y)}{f(x,y) + g(x,y)} \amp= \dfrac{\ds \lim_{(x,y) \to (2,5)} f(x,y)}{\ds \lim_{(x,y) \to (2,5)} \lp f(x,y) + g(x,y) \rp} \amp\amp\text{by Quotient Law}\\ \amp= \dfrac{\ds \lim_{(x,y) \to (2,5)} f(x,y)}{\ds \lim_{(x,y) \to (2,5)} f(x,y) + \ds \lim_{(x,y) \to (2,5)} g(x,y)} \amp\amp\text{by Sum Law}\\ \amp= \dfrac{3}{3 + 7} \\ \amp= \dfrac{3}{10} \end{align*}
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Recall back in MTH 251Z (or MTH 251), the first step to evaluate a limit was to directly substitute the given point into the function. If things worked out, then great we found the limit! If not, then we had to do some some extra math to find the limit.
    ðŸ”—
    This is the same in limits of functions of several variables. If we can directly substitute the point into the function and get a number, then that number is (usually) the limit. If we donâ€™t get a number (especially if we get an indeterminate form like \(\dfrac{0}{0}\)), then we have to do some extra work to find the limit.
    ðŸ”—

    Example 14.2.7.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} \dfrac{\sin\lp x^2 + y^2 \rp}{x^2 + y^2}\text{.}\)
    ðŸ”—
    Hint.
    If the direct substitution doesnâ€™t work, try using a table or a graph to see how the function behaves near the point \((0,0)\text{.}\)
    ðŸ”—
    ðŸ”—
    Solution.
    Letâ€™s try direct substitution first. Observe that
    \begin{equation*} \lim_{(x,y) \to (0,0)} \dfrac{\sin\lp x^2 + y^2 \rp}{x^2 + y^2} = \dfrac{\sin(0)}{0} = \dfrac{0}{0} \end{equation*}
    This is an indeterminate form, so the direct substitution tells us nothing about the limit...
    ðŸ”—
    We can also try evaluating the limit using a table of values. Recall back in MTH 251Z (or MTH 251) that we want to see how the function behaves as we get closer and closer to the point \((0,0)\text{.}\) Then we can make a table of values for the function as BOTH \(x\) AND \(y\) approach 0.
    ðŸ”—
    Observe that the value of \(f(x,y)\) approaches 1 as we get closer to the point \((0,0)\) in any direction. This suggests that
    \begin{equation*} \lim_{(x,y) \to (0,0)} \frac{\sin \lp x^2 + y^2 \rp}{x^2 + y^2}= 1 \end{equation*}
    ðŸ”—
    Alternatively, we can also evaluate the limit using a graph of the function. The graph of \(z = f(x,y)\) is shown below.
    ðŸ”—
    Graphically, we can also see that the value of \(f(x,y)\) approaches 1 as we get closer to the point \((0,0)\) in any direction. This also suggests that
    \begin{equation*} \lim_{(x,y) \to (0,0)} \frac{\sin \lp x^2 + y^2 \rp}{x^2 + y^2}= 1 \end{equation*}
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Unlike graphs of functions of a single variable where we can only approach a point from the left or the right, graphs of functions of several variables allow us to approach a point from infinitely many directions. To argue that a limit exists, we need to show that the function approaches the same value as we approach the point from any direction, like the previous example. If we can find two different directions that give different limit values, then the limit does not exist.
    ðŸ”—

    Example 14.2.10.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} \frac{xy}{x^2 + y^2}\text{.}\)
    ðŸ”—
    Hint.
    If you try direct substitution, you get \(\dfrac{0}{0}\text{,}\) which is an indeterminate form, which means we know nothing about the limits. Making a table of values or looking at graph can be helpful. Richard coded the graph below to help you visualize what the graph looks like.
    ðŸ”—
    What do you expect the limit to be?
    ðŸ”—
    Also, can we come up with an algebraic approach to evaluate the limit without graphing the function?
    ðŸ”—
    ðŸ”—
    Solution 1. When we are looking at the graph...
    Graphically, we can see that the value of \(f(x,y)\) approaches within a range of the \(z\) values between \(-1\) and \(1\text{...}\) We canâ€™t determine a single value that the function approaches as we get closer to the point \((0,0)\text{.}\) Hence, the limit does not exist.
    ðŸ”—
    ðŸ”—
    Solution 2. Using an algebraic approach...
    Without looking at the graph, we can also show that the limit does not exist by showing that the limit along different paths of approach give different values.
    ðŸ”—
    One way of approaching the point \((0,0)\) is along the line \(y = 0\text{.}\) Along this line, we have
    \begin{equation*} \lim_{x \to 0} \dfrac{x \cdot 0}{x^2 + 0^2} = \lim_{x \to 0} 0 = 0 \end{equation*}
    So along the line \(y = 0\text{,}\) the limit is 0.
    ðŸ”—
    Another way of approaching the point \((0,0)\) is along the line \(y = x\text{.}\) Along this line, we have
    \begin{equation*} \lim_{x \to 0} \dfrac{x \cdot x}{x^2 + x^2} = \lim_{x \to 0} \dfrac{x^2}{2x^2} = \lim_{x \to 0} \dfrac{1}{2} = \dfrac{1}{2} \end{equation*}
    So along the line \(y = x\text{,}\) the limit is \(\dfrac{1}{2}\text{.}\)
    ðŸ”—
    Observe that the limit along the line \(y = 0\) is 0, while the limit along the line \(y = x\) is \(\dfrac{1}{2}\text{.}\) Since we get different limit values along different paths of approach, the limit \(\ds \lim_{(x,y) \to (0,0)} \dfrac{xy}{x^2 + y^2}\) does not exist.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    But how do we know if a limit exists without graphing the function? There are thousands of paths of approach to a point... How can we check all of them to see if they give the same limit value?
    ðŸ”—
    One way to improve our efficiency is to check the limit along a family of paths. For example, we can check the limit along ALL the linear paths by setting \(y = mx\) and seeing if the limit depends on the value of \(m\text{.}\) If so, then the limit does not exist.
    ðŸ”—

    Example 14.2.12.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} \dfrac{x^2 - y^2}{x^2 + y^2}\) or show that it does not exist.
    ðŸ”—
    Hint.
    Letâ€™s check the limit along the linear paths. That is, we set \(y = mx\) and see if the limit depends on \(m\text{.}\)
    ðŸ”—
    Can you find two different values of \(m\) that give different limit values?
    ðŸ”—
    ðŸ”—
    Solution.
    Along the linear paths, we can set \(y = mx\text{.}\) That is,
    \begin{align*} \lim_{(x,y) \to (0,0)} \dfrac{x^2 - y^2}{x^2 + y^2} \amp= \lim_{(x,mx) \to (0,0)} \dfrac{x^2 - (mx)^2}{x^2 + (mx)^2} \\ \amp= \lim_{(x,mx) \to (0,0)} \dfrac{x^2 - m^2x^2}{x^2 + m^2x^2} \\ \amp= \lim_{(x,mx) \to (0,0)} \dfrac{1 - m^2}{1 + m^2} \\ \amp= \dfrac{1 - m^2}{1 + m^2} \end{align*}
    ðŸ”—
    This implies that the limit along the linear paths depends on the value of \(m\text{.}\) For example, if \(m = 0\text{,}\) then the limit is \(\dfrac{1 - 0^2}{1 + 0^2} = 1\text{.}\) If \(m = 1\text{,}\) then the limit is \(\dfrac{1 - 1^2}{1 + 1^2} = \dfrac{0}{2} = 0\text{.}\) Since we get different limit values along different paths of approach, the limit does not exist.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    But... if we check the limit along the linear paths and get the same limit value, does that mean the limit exists? The short answer is no. There are other paths of approach than just the linear paths. If we find two different paths of approach that give different limit values, then the limit does not exist.
    ðŸ”—

    Example 14.2.13.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} \dfrac{xy^2}{x^2 + y^4}\) or show that it does not exist.
    ðŸ”—
    Hint.
    Letâ€™s check the limit along the linear paths. That is, we set \(y = mx\) and see if the limit depends on \(m\text{.}\)
    ðŸ”—
    Can you find two different values of \(m\) that give different limit values?
    ðŸ”—
    ðŸ”—
    Solution.
    Along the linear paths, we can set \(y = mx\text{.}\) That is,
    \begin{align*} \lim_{(x,y) \to (0,0)} \dfrac{xy^2}{x^2 + y^4} \amp= \lim{(x,mx) \to (0,0)} \dfrac{x\lp mx\rp^2}{x^2 + \lp mx \rp^4} \\ \amp= \lim_{(x,mx) \to (0,0)} \frac{m^2x^3}{x^2 + m^4x^4} \\ \amp= \lim_{(x,mx) \to (0,0)} \frac{m^2x^3}{x^2(1 + m^4x^2)} \\ \amp= \lim_{(x,mx) \to (0,0)} \frac{m^2x}{1 + m^4x^2} \\ \amp= 0 \end{align*}
    Observe that the limit along all the linear paths is \(0\text{.}\) Does it mean the limit is \(0\text{?}\)
    ðŸ”—
    Richard will throw in another path of approach for us to check. Does the limit along the path \(x = y^2\) also give us \(0\text{?}\)
    ðŸ”—
    We can check quickly that the answer is no. Along the path \(x = y^2\text{,}\) we have
    \begin{align*} \lim_{(x,y) \to (0,0)} \dfrac{xy^2}{x^2 + y^4} \amp= \lim_{(y^2,y) \to (0,0)} \dfrac{y^2 \cdot y^2}{(y^2)^2 + y^4} \\ \amp= \lim_{(y^2,y) \to (0,0)} \dfrac{y^4}{2y^4} \\ \amp= \lim_{(y^2,y) \to (0,0)} \dfrac{1}{2} \\ \amp= \dfrac{1}{2} \end{align*}
    Observe that the limit along the path \(x = y^2\) is \(\dfrac{1}{2}\text{,}\) which is different from the limit along the linear paths. Since we get different limit values along different paths of approach, the limit does not exist.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Hmm... Then how do we know if a limit exists without graphing the function, since there are infinitely many paths of approach to check?
    ðŸ”—
    One way to do so is to use some fancy calculus theorems to argue that there is only one possible limit value that the function can approach as we get closer to the point. One of such theorems is the Squeeze Theorem. In case you donâ€™t recall what the Squeeze Theorem is from MTH 251Z (or MTH 251), below is a quick refresher.
    ðŸ”—

    Theorem 14.2.14. Squeeze Theorem.

    Assume that for \(x \neq c\) (in some open interval containing \(c\)),
    \begin{equation*} l(x) \leq f(x) \leq u(x) \qquad \text{ and } \qquad \lim_{x\to c}l(x) = \lim_{x \to c} u(x) = L \end{equation*}
    Then \(\ds \lim_{x\to c} f(x)\) exists and \(\ds \lim_{x\to c} f(x) = L\text{.}\)
    ðŸ”—
    ðŸ”—
    A classic application of the Squeeze Theorem is to evaluate limits of functions involving some trigonometric functions like sine and cosine, since they are bounded between \(-1\) and \(1\text{.}\) So we can often convert the function to polar coordinates and try using the Squeeze Theorem to evaluate the limit.
    ðŸ”—

    Example 14.2.15.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} \dfrac{x^2 - y^2}{\sqrt{x^2 + y^2}}\text{.}\)
    ðŸ”—
    Hint.
    Letâ€™s convert the function to polar coordinates. That is, we want the variables \(r\) and \(\theta\) instead of \(x\) and \(y\text{.}\)
    ðŸ”—
    Then try using the Squeeze Theorem to evaluate the limit.
    ðŸ”—
    P.S. If you forgot how to do so, find some clues in SectionÂ 12.7.
    ðŸ”—
    ðŸ”—
    Solution.
    Recall that \(x = r\cos(\theta)\) and \(y = r\sin(\theta)\text{.}\) Then we obtain
    \begin{align*} \dfrac{x^2 - y^2}{\sqrt{x^2 + y^2}} \amp= \dfrac{r^2\cos^2(\theta) - r^2\sin^2(\theta)}{\sqrt{r^2}} \\ \amp= r\lp \cos^2(\theta) - \sin^2(\theta) \rp \\ \amp= r\cos(2\theta) \end{align*}
    ðŸ”—
    Clearly, if \((x,y) \to (0,0)\text{,}\) then \(r \to 0\text{.}\) So the problem becomes
    \begin{equation*} \lim_{(x,y) \to (0,0)} \dfrac{x^2 - y^2}{\sqrt{x^2 + y^2}} = \lim_{r \to 0} r\cos(2\theta) \end{equation*}
    ðŸ”—
    Since \(\cos(2\theta)\) is bounded between \(-1\) and \(1\text{,}\) we have
    \begin{equation*} -r \leq r\cos(2\theta) \leq r \end{equation*}
    As \(r \to 0\text{,}\) both bounds approach zero. By the Squeeze Theorem, we conclude that
    \begin{equation*} \lim_{r \to 0} r\cos(2\theta) = 0 \end{equation*}
    Therefore,
    \begin{equation*} \lim_{(x,y) \to (0,0)} \dfrac{x^2 - y^2}{\sqrt{x^2 + y^2}} = 0 \end{equation*}
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Just a quick note that we donâ€™t have to convert the function to polar coordinates to use the Squeeze Theorem if you can find some nice bounds for the function in terms of \(x\) and \(y\text{.}\) This usually happens when the function involves some trigonometric functions like sine and cosine, or other functions that are bounded between two numbers.
    ðŸ”—

    Example 14.2.16.

    Evaluate the limit \(\ds \lim_{(x,y) \to (0,0)} 3x^2 \sin\lp \dfrac{1}{x^2 + y^2} \rp\text{.}\)
    ðŸ”—
    Hint.
    You are more than welcome to try the direct substitution first, but you will get an indeterminate form... So inconclusive!
    ðŸ”—
    Alternatively, you can also try finding two paths of approach that give different limit values, good luck!
    ðŸ”—
    As a third option, you can also try using the Squeeze Theorem since there is a very obvious sine function in the expression.
    ðŸ”—
    ðŸ”—
    Solution.
